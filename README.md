# Terminal User Interface - Hello World
* ich spiele ein bisschen rum und möchte lernen TUI (Terminal User Interfaces) zu bauen.

## Genutzt wird
* C++ (20)
* Cmake
* [FTXUI](https://github.com/ArthurSonzogni/FTXUI)

## install
```bash
# Für Arch - Linux

sudo pacman -S yay
yay -S ftxui
```

## Use / Build Projekt
```bash
mkdir build
cd build
cmake ..
make
./tui_helloworld
```
